# Simple Interest Calculator

A simple yet effective tool to calculate the interest earned or paid on a principal amount over a specific period at a given annual interest rate. This project is part of a foundational exercise in web development and cloud computing.

- **User-friendly Interface:** Easy to input data and get instant results.
- **Accuracy:** Precise calculations based on standard financial formulas.
- **Responsive Design:** Works smoothly on desktops, tablets, and mobile devices.

## 🧮 How It Works
The calculator takes three main inputs from the user:
1. **Principal:** The initial amount of money.
2. **Annual Interest Rate:** The percentage of interest charged/earned per year.
3. **Period:** The time in years for which the money is borrowed or invested.

## 📐 Formula
The calculation is based on the following standard simple interest formula:

$$I = \frac{P \times r \times t}{100}$$

Where:
- **I** = Simple Interest
- **P** = Principal amount
- **r** = Annual interest rate (in %)
- **t** = Time period (in years)

## 🚀 Usage
1. Open the project in your browser.
2. Enter the **Principal** amount.
3. Enter the **Interest Rate** per year.
4. Select the **Number of Years**.
5. Click the "Calculate" button to see the result.
